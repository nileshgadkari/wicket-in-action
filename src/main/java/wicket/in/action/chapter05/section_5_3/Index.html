5.3 Navigating using links

These examples show how to navigate using components.

5.3.1 Linking to documents using static links

This example shows how to link with static links inside the markup.

The markup in the following example

<!-- markup -->
<a href="http://manning.com/dashorst">Wicket in Action</a>

becomes:

Wicket in Action

5.3.2 Using ExternalLink to render links programmatically

This example shows how to an external link.

The following markup and Java code

<!-- markup -->
<a href="#" wicket:id="recipe">foobar</a>

/* Java code */
add(new ExternalLink("recipe", recipe.getUrl(), recipe.getName()));

Becomes:

foobar

5.3.3 Linking to Wicket pages with BookmarkablePageLinks

This example shows how to a bookmarkable page link.

The following markup and Java code:

<!-- markup -->
<a href="#" wicket:id="book">Book</a>

/* Java code */
add(new BookmarkablePageLink("book", Book.class));

becomes:

Book

The following example shows how to a bookmarkable page link together with PageParameters.

<!-- markup -->
<a href="#" wicket:id="details">Details</a>

/* Java code */
add(new BookmarkablePageLink("details", CheeseDetailsPage.class, 
                         new PageParameters("name="+cheese.getName()));

becomes:

Details

5.3.4 Adding bookmarkable links automatically with wicket:link

The following example shows how to a create a simple menu using <wicket:link>.

<wicket:link>
    <ul>
        <li><a href="Index.html">This page</a></li>
        <li><a href="Page1.html">Page1</a></li>
        <li><a href="Page2.html">Page2</a></li>
    </ul>
</wicket:link>

Becomes:

  • This page
  • Page1
  • Page2
