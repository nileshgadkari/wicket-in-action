5.4 Responding to client actions with a link

These examples show how to links as interactive components.

5.4.1 Using Link to respond to client actions

This example shows how to use a Link as an interactive component.

The link in this example navigates to another page in its onClick handler.

<!-- markup -->
<a href="#" wicket:id="link1">click me</a>

/* Java code */
add(new Link("link1") {
    @Override
    public void onClick() {
        // do something special
        Page next = new Page1();
        setResponsePage(next);
    }
});

becomes:

click me
